# TRAKTFETE

A lightweight and minimal spending tracker that is designed to be as simple as possible.

## Features

- Log daily spending and track them through a calendar type system
- Limit how much you spend through the week with the help of a spending limit that resets every week
- See where your money is disappearing to by showing you a weekly report of your spendings
- Locally saved to your device the make the app free forever!
